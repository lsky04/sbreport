📥 거래내역 가져오기
은행 거래내역 CSV를 업로드하면 수입/지출을 자동 분류합니다. 국민·신한·하나·우리·기업·농협·카카오 형식 지원.
🏦
거래내역 CSV 파일을 드래그하거나 클릭해서 업로드
은행 앱/홈페이지 → 거래내역 조회 → 엑셀/CSV 다운로드
엑셀(.xlsx)은 먼저 다른 이름으로 저장 → CSV(쉼표로 분리)로 변환 후 업로드
🏦 국민은행🏦 신한은행 🏦 하나은행🏦 우리은행 🏦 기업은행🏦 농협 🏦 카카오뱅크
💡 CSV 준비 방법
  ① 은행 사이트에서 거래기간 설정 후 엑셀/CSV로 내보내기
  ② 엑셀 파일이라면: 파일 → 다른 이름으로 저장 → CSV(쉼표로 분리)(*.csv) 선택
  ③ 저장된 .csv 파일을 여기에 드래그 or 클릭해서 업로드

⚙️ 자동 분류 키워드 설정

거래 적요(메모)에 아래 키워드가 포함되면 해당 분류로 자동 지정됩니다.
우리 회사에 맞게 키워드를 추가·삭제해서 정확도를 높이세요.
